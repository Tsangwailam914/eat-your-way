## How It's Made:

**Tech used:** EJS, SCSS, JavaScript, MVC

## Optimizations

## Lessons Learned:

## Installation:
1. Clone the repo
2. Run `npm install` for the modules

## Usage:
1. Run `npm run dev`
2. Go to http://localhost:2121
3. Add your own DB string in the env file.
